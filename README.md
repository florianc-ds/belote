## DESCRIPTION
This project, developped in React, proposes a belote game.

It makes you possible to choose the levels of your partner and your opponents.
It works along with the [belote_api](https://github.com/florianc-ds/belote_api) project that implements agents to play with.


## LAUNCH LOCALLY
`yarn start`


## PRODUCTION USAGE
Build the app

`yarn build`

Serve the application

`serve -l <port> -s build`

**N.B:** Production build differ from local mode in following points:
- **source map** will not be generated
- **React Developper Tools** will be disabled
